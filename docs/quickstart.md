# Fabric 快速入口

`docs/getting-started.md` 现为 Fabric v1.0 的标准上手入口。

完整 7 阶段路径请见 [Getting Started](./getting-started.md)：

1. 安装 Fabric
2. 准备目标项目
3. 运行 `fab init`
4. 配置 agents 与 hooks
5. 启动 `fab serve`
6. 确认 MCP 已激活
7. 记录首条 ledger entry

本页仅作为既有链接的兼容重定向。`FAB_SERVER_PATH` 设置与面向贡献者的本地开发说明现位于 [Contributing](./contributing.md)。
