# Fabric 快速入口

`docs/getting-started.md` 是 Fabric 当前的标准上手入口。

如果你只想先走最短路径，请按下面的顺序做；完整说明见 [Fabric 上手](./getting-started.md)：

1. 安装 Fabric
2. 准备目标项目
3. 运行 `fabric init`（一站式完成基础文件、bootstrap、MCP 和 hooks）
4. 验证规则文件与 hooks
5. 启动 `fabric serve`
6. 确认 MCP 已激活
7. 记录首条 ledger entry

本页只保留最短路径。`FAB_SERVER_PATH` 和面向贡献者的本地开发说明见 [Contributing](./contributing.md)。
