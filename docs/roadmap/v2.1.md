# v2.1 Roadmap

> Authority: `docs/design/v2.1-knowledge-schema-redesign.md` (设计稿)
> Memory: `memory/project_l0_l1_l2_redesign_v21.md` (共识)

v2.1 = knowledge schema redesign 整体落地。SDK breaking 级别 + 跨包重命名 + disk migration,拆 3 个 rc 滚动,每 rc 各自发 maestro-plan 独立 plan。

**Knowledge schema 正交化** — L0/L1/L2 enum 整体删,改 `scope` (team/personal) + `relevance` (path/always) 正交两字段,根治 v1→v2 残留的 layer 词义污染 + 3 档过度设计。

## 中止/回退条件

任一条触发,v2.1 redesign abort 回退 v2.0 schema:
- rc.1 dogfood 发现 MCP tool 重大行为 regression
- rc.2 自身 migrate 失败或 entry 内容损坏
- doctor `--migrate-v21` failure rate > 5% per-entry
- 第三方 fabric-server SDK 用户(若有)反馈无法升级

回退路径:revert v2.1-rc.* commits,publish v2.0.0 stable(从 rc.30 升级),redesign 推迟 v3.0 重启。
