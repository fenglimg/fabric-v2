# Handoff: Fabric Init Bootstrap Internalization

Date: 2026-04-22
Branch: `main`

## User Goal

Continue the Fabric init UX/bootstrap optimization and change the target behavior so `fab init` / `fab bootstrap install` no longer expose root-level bootstrap docs:

- Do not generate root `AGENTS.md`.
- Do not generate root `CLAUDE.md`.
- Do not generate root `GEMINI.md`.
- Keep the visible bootstrap artifact at `.fabric/bootstrap/README.md`.
- Commit and push all current task artifacts.
