# werewolf-minigame-stub

Minimal Cocos Creator fixture for demonstrating Fabric multi-agent collaboration.
The sample now includes five-role `.fabric/agents/` shadow constraints for villager, werewolf, seer, witch, and hunter coordination.
The root `AGENTS.md` is intentionally only a Fabric Bootstrap Protocol stub.
Use it as a lightweight documentation example or as a disposable target when validating Fabric flows from this monorepo.
If you need to validate `fab init` against an empty target, copy this fixture to a temporary directory first and remove any pre-existing Fabric artifacts before running the command.
